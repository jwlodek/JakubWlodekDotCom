Welcome to my website!

Hello! My name is Jakub Wlodek, and I am a Computer Science and Applied Mathematics double major at Stony Brook University, currently in my senior year. I work at Brookhaven National Laboratory as a student assistant for the controls group at the National Synchrotron Light Source II. My work primarily revolves around writing Computer Vision based tools for scientists, along with developing new plugins and detector drivers for the open source area detector project. Make sure to visit my Github for source code and more information about my projects!

Projects

3D Focus Stacking


Worked on 3D reconstruction software using focus stacking with python, OpenCV, and OpenGL. Software used image alignment, edge detection, and thresholding to generate a merged image from a stack of images, along with a depth map. These were then used to generate a 3D model with OpenGL. Videos of 3D models can be seen on youtube and more information about the project can be found in the publication linked above.

ADPluginBar


As a way to simplify sample managment at National Synchrotron Light Source II beamlines, I worked on constructing a barcode reader plugin for the area detector project (A link to the areaDetector github page can be found in the footer of this site). The plugin integrated the opencv and zbar libraries for image processing and barcode detection respectively, and wrote detected barcode messages into Process Variables, or PVs, which were then inserted into a database. Source code is available on my Github account.

Rowers Meet


For my principles of database systems course at Stony Brook University, I worked as part of a small team to create a dating website using HTML, CSS, Javascript, and Bootstrap for the front end portion, and Java and MySQL for the website backend. We used .jsp files and servlets to integrate the html with the output of the jdbc based SQL queries, and we hosted the completed website on an Apache server. Source code for the project is available on my Github account.

ADCompVision


Using the experience I gained when writing the ADPluginBar reader and the 3D image stacking software, I began constructing an implementation of core OpenCV Computer Vision functions for area detector in the form of a plugin. Some currently added functionality is edge detection and centroid detection. The initial testing of the plugin involved computing the centroid of samples on a stand in order for automation to be able to align sample centroids. Source code for the entire plugin is available on Github.
