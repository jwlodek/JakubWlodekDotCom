About Me!

Welcome to my website! My name is Jakub Wlodek, and I am a Graduate student for Computer Science at Stony Brook University.

This year, I graduated with a CS and Applied Math major and U.S. history minor, and I chose to continue with graduate studies to finish my Master's degree in one year. I currently work at Brookhaven National Laboratory as a junior engineer for the controls group at the National Synchrotron Light Source II.

My work primarily revolves around contributing Computer Vision based tools and drivers to the open source areaDetector project many of which are at use in production on NSLS2 beamlines. The ADUVC areaDetector driver I developed for USB Video Class cameras has been deployed on three beamlines, and provides scientists a cost-effective solution for monitoring cameras. The 17BM beamline at NSLS2 currently uses ADUVC based cameras as their primary acquisition hardware, with 7 currently deployed.

The ADCompVision plugin I wrote that integrates real time OpenCV image processing for all areaDetector supported cameras has been deployed on several beamlines, and has been used for robotic alignment, distance measurement, and edge detection. One other application for ADCompVision was to enable a dual threshold feature for a specific X-Ray detector that required consecutive images from the camera to be subtracted in real time, with one representing the upper threshold, and one the lower threshold.

In addition to working on areaDetector, I am also focused on improving the deployment of control software through deployment systems and scripting. This work primarily involves some custom build scripting along with utilizing Docker containers for building accross all platforms.

If you would like to learn more about these projects, make sure to visit the Projects page, and visit github for source code. Outside of work and school, I have a passion for all things technological, and I also greatly enjoy spending time outside or on the water. In fact, I even row on the Stony Brook University Crew team.


Projects

More details regarding each of these projects can be found under the "Projects" tab, or by clicking a link below.

  • py_cui - A python library for constructing widget-based, ascii-only command line interfaces.
  • csplit - A single header C library to easily handle splitting and processing strings.
  • ADCompVision - A C++ real-time OpenCV based computer vision plugin for the areaDetector project.
  • Focus Stacking - A program that uses OpenCV and OpenGL to perform focus stacking and 3D reconstruction.
  • SAVJ-Canvassing - A Java and MySQL backed web app for political canvassing.
  • ADUVC - A C and C++ areaDetecor driver that adds support for USB Video Class (webcam) devices to EPICS.
  • installSynApps - A python module and Dockerfiles that allow for automatic configuration, compilation, and packaging of EPICS and areaDetector.
  • Through The Ages - A web based game written in javascript with the help of the phaser game engine.
  • Rowers Meet - A web app built with Java and MySQL that creates a dating website for college Rowers
  • Foldify - An in-progress C# WPF desktop app for file management and organization.
  • ADPluginBar/Dmtx - A QR/Bar/Data Matrix code reader for EPICS areaDetector supported cameras.
  • Other - My many other projects
