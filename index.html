About Me!

Welcome to my website! My name is Jakub Wlodek, and I am a Graduate student for Computer Science at Stony Brook University. This year, I graduated with a CS and Applied Math major and U.S. history minor, and I chose to continue with graduate studies to finish my Master's degree in one year. I work at Brookhaven National Laboratory as a junior engineer for the controls group at the National Synchrotron Light Source II. My work primarily revolves around writing Computer Vision based tools for scientists, along with developing new plugins and detector drivers for the open source area detector project. I am also focused on imporving the deployment of control software through deployment systems and scripting. If you would like to learn more about these projects, make sure to visit the "Projects" page, and visit github for source code. Outside of work and school, I have a passion for all things technological, and I also greatly enjoy spending time outside or on the water. In fact, I even row on the Stony Brook University Crew team.


Projects

A more expansive and detailed list of my projects can be found under the "Projects" tab.


  • Focus-Stacking - A program that uses OpenCV and OpenGL to perform focus stacking and 3D reconstruction.
  • Foldify - An in-progress C# WPF desktop app for file management and organization.
  • ADCompVision - A computer vision plugin for the areaDetector project.
  • ADUVC - An areaDetecor driver for USB Video Class devices.
  • installSynApps - A collection of python and shell scripts that allow for automatic configuration and compilation of EPICS and areaDetector.
  • libCGeo - An in-progress C library for computational geometry.
  • Through-The-Ages - A web based game written in javascript with the help of the phaser game engine.
